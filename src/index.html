      Datenkammer

      Informationen zur Datenkammer

      Hier können Sie strukturierte Daten zu Leads erfassen. Jedes Datenpaar besteht aus einem Schlüssel (z.B. "email") und einem Wert.

      Die Daten werden mit dem aktuellen Chat verknüpft und können später exportiert oder in der Datenbank gespeichert werden.

      Bestehende Lead-Daten werden durch neue Einträge ergänzt, doppelte Schlüssel werden überschrieben.
